



# SolonCodeCLI

SolonCodeCLI 是一个基于 Solon AI （Java）框架实现的 AI 智能体应用。它参考 Claude Code CLI 实现，旨在为开发者提供一套高效、轻量且高度可扩展的本地 AI 辅助开发工具。

* 兼容 java8 ~ java25 运行环境
* 兼容  Claude Code Agent Skills 规范
* 支持 cli 命令行体验
* 支持 webapi 体验
* 支持 acp 协议（可接入支持的 IDE）
